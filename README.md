# blog
개인 블로그 애플리케이션 입니다.

* 게시글 : 메뉴별 게시글 노출
* 태그 : 메뉴별 태그를 통한 카테고리 검색
* 검색 : 검색을 통한 게시글 조회
* 어드민 : otp 인증, 비밀번호를 통한 어드민 툴 접속.
  * 글쓰기 : 각 항목 입력 및 이미지 업로드, 마크다운을 통한 글쓰기 지원
  * 소개글 : 이름 및 마크다운을 통한 소개글 수정 지원
  * 메뉴 추가 : 새로운 메뉴 등록 지원
